# 📊 Sales Data Analysis

## 📌 Overview

This project analyzes sales data to identify trends, profitability, and business opportunities.

## 🛠️ Tools Used

* Python (pandas, matplotlib)
* Jupyter Notebook

## 📊 Key Analysis

* Sales by category
* Profitability analysis
* Monthly sales trends

## 🔍 Insights

* High sales do not always mean high profit
* Some categories perform better than others
* Sales show variability over time

## 💡 Business Recommendations

* Focus on high-margin products
* Improve low-performing categories
* Use data-driven strategies

## 📁 Project Structure

* `analysis.ipynb` → main analysis
* `images/` → visualizations
* `dataset` → sales data

## 🚀 Outcome

This project demonstrates data analysis, visualization, and business thinking skills.
